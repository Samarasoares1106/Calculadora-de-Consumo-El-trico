# 🔋 Calculadora de Consumo Elétrico ⚡

Uma ferramenta simples para estimar o consumo de energia elétrica de aparelhos domésticos.

---

## Objetivo 🎯
Ajudar o usuário a calcular o consumo mensal de energia de um aparelho e estimar o custo aproximado.

---

## Linguagem 🐍
- Python 3

---

## Fórmula usada 🧮
consumoMensal = (potencia * horasDia * 30) / 1000

---

## ▶️ Como executar o projeto

# Clone este repositório
git clone https://github.com/samarasoares1106/consumo-energia.git

# Acesse a pasta do projeto
cd consumo-energia

# Execute o programa
python app.py

---

## 🛠️ Requisitos
- [x] Ter o **Python 3** ou superior instalado em sua máquina.
- [x] Um terminal ou prompt de comando para execução.

---

## ✨ Funcionalidades
* **Cálculo de Consumo:** Descubra o consumo mensal exato em kWh ⚡
* **Estimativa de Custo:** Saiba quanto vai pagar em reais (R$) 💰
* **Interface Simples:** Experiência amigável via linha de comando 👌
